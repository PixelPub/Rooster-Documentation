Initial Setup

When you are ready to install a theme, you must first upload the theme to your WordPress directory. There are two ways you can install the theme:

  • Using FTP: By using any FTP program of your choice you can upload the non-zipped folder to the /wp-content/themes/ folder on your server.
  • Using WordPress: Navigate to Appearance → Themes → Install Themes. Go to upload and select the zipped theme folder and hit install now and theme will be uploaded and installed. Once you have uploaded the theme, you need to active it. To activate your new theme, go to Appearance → Themes and activate your chosen theme.

After you installed the theme you need to install the "Rooster Helper" plugin. You see an info panel with a direct link to install it in your admin backend.

That's all, you're done installing!

Import Demo Content

If you prefer to start with existing content you can import our demo content. Simply download this zip file and import it into your wordpress. To import use "Tools->Import".

Theme features

This theme makes extensive use of the WordPress Customizer. Use the "Customize" link under the "Appereance" tab. It features several sections:

  • Site Title and Tagline

    Your Site Title will appear in the Browser Tab. The tagline is not used by this theme.

  • General
    • Background Image

      If set the background will be filled with this image

    • Background Color

      If no background image is set, this color will be used

    • Content Background

      Defines the background color of the main content area

    • Divider

      Sets the color for all dividers, menu, menu-hover effect and navigation elements

    • Font

      Defines the font color for continous text

    • Post/Project Title

      Defines the title color for blog posts and projects on detail and overview pages

    • Button

      Defines the color for all buttons

    • Button Text

      Defines the text color of all buttons

  • Blog
    • Author

      Defines the color of the author of blog entries on overview and detail page

    • Blog Meta Info

      Defines the color of the date information, of blog entries on overview and detail page

  • Projects
    • Project Meta Heading

      Defines the color of the headings for project meta information (Client, Project Date, Skills)

    • Project Meta

      Defines the color of the content for project meta information (Client, Project Date, Skills)

  • Comments
    • Textarea

      Defines the background color of the textarea for comments

  • Contact

    These settings are primarly used on the contact sub page. They should be fairly self explanatory.

    • Company
    • Address
    • Phone
    • Mobile
    • Maps

      Put your google maps embed URL here. Dont paste the complete iFrame code. This means only the value of the src="URL" of your embed code.

    • Mail
  • Header Image

    Defines the logo. Your logo is supposed to be 300x70 pixels in size. You can crop your image when uploaded. But basically you can use whatever size you want.

  • Footer
    • Color

      Defines the color of footer elements on the left and right side

    • Left Text

      Defines the text which is placed in the bottom left corner

  • Social

    Here you can set your usernames for various social platforms.

    • Icons

      Defines the color of all social icons that appear on the page

    • Various social platforms

      Defines the text which is placed in the bottom left corner

  • Navigation
    • Header Menu

      Defines which menu to use in the header

    • Footer Menu

      Defines which menu to use in bottom right corner

    • Font

      Defines the color of all navigation fonts

  • Static Front Page

    This lets you decide which page you want to have as default start page.

Setting up the pages

Home

The styling of your home page is up to you. Just choose "Default template"

Portfolio

You can choose between 3 pre defined layouts. They will show your portfolio as one, two or three column grid respectively. If you want to use them elsewhere, you can use our shortcodes. See the "Shortcodes" section for further reference.

Contact

To get a contact site like in the demo simply choose "Contact" as page template. Insert your personal information via the Customizer as described above.

About

The about page displays your custom content from the editor and all of your team members. You can add them via the "Team" tab in the admin panel.

Blog

Projects

You can add projects via the tab "Portfolio" in the admin panel. Projects feature the following information.

  • Featured Image

    This image will be shown in the top area. A filter will be applied to the image.

  • Project URL
  • Project Date
  • Skills
  • Client

Team

Your team members can be added via the "Team" tab in the admin panel. They consist of a custom text and a featured image. Team members will be shown in the bottom of the "About" template.

Shortcodes

We have several shortcodes for you.

Portfolio

This shortcode generates a portfolio grid.

Attributes

  • cols - How many colums to produce (default: 2)

Example

[portfolio cols="1"]

Maps

This shortcode generates a google maps iFrame.

Attributes

  • width - (default: 100%)
  • height - (default: 600)
  • frameborder - Whether the iFrame displays a border or not (default: 0)
  • style - additional styling (default: "border:0")
  • src - The embed code URL (default: your settings in the customizer)

Example

[maps width="300px" height="300px" frameborder="1" style="border:0"]

Tagline

This shortcode display a full width tagline.

Attributes

  • none

Example

[tagline] I really want you to read this! [/tagline]

Columns

These shortcodes generate columns with 1/2, 1/3 or 1/4 size. Be sure to always fill up to a whole row. Do not just use a single "one_half" shortcode!

Attributes

  • first - "yes"/"no" specifies whether this is the first column (default: '')
  • last - "yes"/"no" specifies whether this is the last column (default: '')

Example

[one_half first="yes"] This is my first half [/one_half]
[one_half last="yes"] This is my second half [/one_half]

[one_third first="yes"] This is my first third [/one_third]
[one_third] This is my second third [/one_third]
[one_third last="yes"] This is my last third [/one_third]

[one_fourth first="yes"] This is my first fourth [/one_fourth]
[one_fourth ] This is my second fourth [/one_fourth]
[one_fourth ] This is my third fourth [/one_fourth]
[one_fourth last="yes"] This is my last fourth [/one_fourth]
